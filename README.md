# 💼 CV Web - Rocio Abril Alaniz

Este proyecto es un **Currículum Vitae (CV) técnico online**, desarrollado con **HTML y CSS**, donde presento mi perfil profesional, habilidades y proyectos.

---

## 📌 Descripción

Se trata de una página web estática que funciona como mi CV digital.  
El objetivo es mostrar de forma clara y ordenada:

- Mi perfil profesional  
- Tecnologías que manejo  
- Proyectos realizados  
- Información de contacto  

Este proyecto está pensado para complementar mi perfil profesional y compartirlo con reclutadores.

---

## 🛠 Tecnologías utilizadas

- HTML
- CSS  

---

## ✨ Funcionalidades

- 📄 Presentación de perfil profesional  
- 🧠 Lista de tecnologías y habilidades  
- 🚀 Sección de proyectos con enlaces externos  
- 📬 Información de contacto
